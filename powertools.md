The makerspace is equiped with a few power tools to help with prototyping and manufacturing, but these power tools are dangerous. If you are uncertain how to use a specific power tool, do not guess. Ask for assistance from someone with experience or ask in the makerspace slack channel.

# Power tool safe usage guidelines
1. Always have someone else present in the room when using the Band Saw, Laser Cutter, Drill Press or Dremmel.
2. Always wear apropriate PPE.
3. Always tie back long hair.
4. Never wear gloves when using the Band Saw - it can make injuries much worse.
5. Never use power tools you are not confident with. Ask for assistance.
